# 学习星球

**从小学到大学的知识点导航**

一个开源的中国教育知识点整理项目，覆盖 10 个学科、4 个学段，目标是构建一个完整、准确、易于浏览的知识体系地图。

## 项目缘起

中国教育体系涵盖小学到大学十余年的学习历程，知识面广、学科繁多。然而，目前缺乏一个**完整的、结构化的、免费的**知识点索引工具。

这个项目的出发点很简单：

- **梳理知识脉络**：把零散的知识点串联成体系，帮助学习者建立全局视角
- **降低学习门槛**：用清晰的结构和简洁的语言呈现每个知识点
- **开源共建**：知识不应被围墙圈住，期待更多人参与完善

### 已知不足

以下是项目目前存在的不足，也是我们需要帮助的方向：

**内容方面**：
- 部分学科知识点数量偏少（物理 16 个、生物 16 个），需要补充课内完整内容
- 大学阶段仅覆盖基础课程，专业课内容几乎为空白
- 缺少练习题和真题链接
- 部分知识点的例题和公式可以更丰富
- 缺少知识图谱（知识点之间的关联关系尚不完整）

**功能方面**：
- 当前为纯静态页面，无后端支持
- 缺少用户收藏、笔记功能
- 缺少知识点掌握进度追踪
- 搜索功能为前端模糊匹配，可以更智能
- 移动端体验有待优化

**技术方面**：
- 所有数据存储在一个 JS 文件中（300KB+），数据量大时需考虑数据库方案
- 缺少自动化测试
- 缺少构建工具和部署流程

## 如何参与

我们欢迎各种形式的贡献！无论你是学生、老师、还是对教育感兴趣的开发者，都可以参与：

### 贡献知识点

最需要的帮助是**完善知识点内容**。每个知识点包含以下结构：

```javascript
{
  id: 'sx-001',           // 唯一标识
  order: '01-02-001',     // 排序编号（学段-学科-序号）
  subject: '数学',         // 学科
  stage: '小学',           // 学段
  title: '加减法运算',     // 标题
  description: '...',      // 简短描述
  tags: ['运算', '基础'],  // 标签
  difficulty: 1,           // 难度（1-5）
  detail: {
    concept: '...',        // 概念（核心知识内容）
    formula: '...',        // 公式/定理
    examples: [...],       // 例题
    tips: '...',           // 易错点/学习提示
    related: [...]         // 关联知识点
  }
}
```

**当前最需要补充的方向**：
- 物理和生物的课内知识点
- 大学阶段的专业课程内容
- 每个知识点的练习题和真题
- 错误纠正和内容校对

### 贡献代码

- 功能开发：搜索优化、进度追踪、移动端适配
- 数据管理：将 JS 文件迁移到数据库
- UI/UX：页面设计改进、交互优化
- 自动化：CI/CD、数据校验脚本

### 参与方式

1. Fork 本仓库
2. 创建你的功能分支（`git checkout -b feature/your-feature`）
3. 提交改动（`git commit -m 'Add some feature'`）
4. 推送到分支（`git push origin feature/your-feature`）
5. 提交 Pull Request

### 内容规范

- 使用**准确的教材用语**，避免口语化表述
- 公式和定理需要标明来源和适用条件
- 例题要由浅入深，附简要解析
- 易错点要写出常见的错误和正确做法
- 难度分级：1（基础）- 2（理解）- 3（应用）- 4（拓展）- 5（竞赛）

## 技术栈

- 纯前端实现：HTML + CSS + JavaScript
- 无框架依赖，零构建步骤
- 直接用浏览器打开 `index.html` 即可运行
